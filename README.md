# Additional Results for HDTE-NN

This repository contains additional visualizations supporting the interpretability analysis of the proposed HDTE-NN estimator.  
All results are averaged over 100 repeated sample splits.

---

# Dataset 1: NHANES

## 1. Overall average treatment-effect curves

![Overall curves](plot5_practical_overall_curves.png)

*Overall average estimated treatment-effect curves for the NHANES data as a function of the BMI threshold, averaged over 100 repeated sample splits.*

---

## 2. Subgroup-average treatment-effect curves

![Subgroup curves](plot6_practical_subgroup_curves.png)

*Subgroup-average estimated treatment-effect curves for the NHANES data, shown for age, child sex, and household income, averaged over 100 repeated sample splits.*

---

## 3. Representative-profile treatment-effect curves

![Representative profiles](plot7_practical_representative_profiles.png)

*Estimated treatment-effect curves for representative covariate profiles in the NHANES data corresponding to the 0.10, 0.50, and 0.90 quantiles of the age distribution, averaged over 100 repeated sample splits.*
